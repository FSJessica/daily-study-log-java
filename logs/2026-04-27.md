##  🔹 Feito

## 🔹 Aprendi

- É importante customizar essas duas interfaces em sistemas reais,
pois o Spring fornece uma estrutura básica e genérica que pode não atender
a uma situação real.
- Customizar UserDetails para representar melhor os dados importantes,
customizar UserDetailsService para definir como o Spring deve encontrar esse
usuário no banco de dados.
